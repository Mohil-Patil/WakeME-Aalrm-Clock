# wakeME-Aalrm-Clock
WakeME Alarm clock is an Android Alarm Cock Application with a 'Solve Math' Feature that requires the User to solve the addition of two displayed numbers if the user enters the correct answer the alarm stops ringing but if the user enters the wrong answer alarm will keep ringing until the correct answer is entered.
It helps overcome laziness and increases the productivity of the user.
